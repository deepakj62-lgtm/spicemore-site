🚀 One-tap path
Tap any page below. On iPhone/iPad with the Claude app installed, it opens the app directly. Otherwise it opens claude.ai in your browser. Either way — Claude is pre-loaded with the system rules and the current source for that page.
Tip: First time you tap, your phone may ask whether to open in Claude or in browser. Pick Claude. After that it remembers.
What happens after you tap: Claude reads the system rules + current source code, then says "Got it — what would you like to change?". Type your change in plain English and watch the preview update.
⚡ Quick path
Three steps. No account upgrade. Works in the Claude iOS app, web, or anywhere.
Step 1 — pick which page you want to edit
Step 2 — copy this prompt
Tap the button. It copies a complete prompt (system context + the chosen file's source) ready to paste into Claude.
Step 3 — paste into Claude and chat
  1. Open the Claude app (or claude.ai in a browser).
  2. Start a new chat. Paste.
  3. After the prompt is in, type what you want to change. Examples:
    • "Add a checkbox so staff can mark whether the client brought their PAN card."
    • "Change the Create Intake button text to 'Confirm and Create'."
    • "Make the warning above Create Intake red instead of yellow."
  4. Claude shows an artifact preview of the page — tap it to see the full page render live. Iterate as long as you want.
  5. When happy, say: "I'm done — handoff." Claude outputs the final file + a short summary.
  6. Send the artifact / summary to Deepak (WhatsApp, email, or share the Claude conversation link). He'll ship it to spicemore.com (live in ~60 sec).
📁 Project path
Persistent workspace. The source files live in your Project Knowledge — Claude reads them on every chat. ~5 min to set up.

  1. Open claude.ai on your iPad or laptop.

  3. Tap Projects → + New project. Name it: Spicemore SEPL Pages.

  4. Send that summary to Deepak. He'll ship it.
📨 Sending changes back to Deepak
Three ways. Pick whichever is easiest.
  • WhatsApp — copy the artifact text or share the Claude chat URL. Send to Deepak. Fastest.
  • Email — to deepakj62@gmail.com with subject prefix SEPL HANDOFF:.
  • Shared Drive folder — if you've been added to the iCloud / Drive folder, drop the .html file there.

Deepak's Claude Code reads the handoff, sanity-checks against the system rules, commits, and pushes to GitHub. Vercel auto-deploys to spicemore.com. Live in ~60 sec.

What Claude can change here (and what it can't)
Can change: any visual element on the SEPL pages — text, layout, colours within the Spicemore palette, button positions, dropdown options, validation messages, mandatory-field markers, new fields, new sections, new tabs, calculation labels.

Can't change directly (these need Deepak's Claude Code): backend API endpoints, database, scrapers, WhatsApp delivery, OTP, agreement PDF text, LTV bands, margin call thresholds, deployment/infra. If you ask for any of these, Claude will flag "backend change needed — Deepak handles."
Pricing / plan notes
Both paths work on Free, Pro, and Team Claude plans. Free plan has daily message limits (~5–10 messages); Pro is ~5× higher. For day-to-day SEPL editing, Free is enough; for heavy iteration, Pro at $20/month is comfortable.

Claude Projects ARE per-account — they don't auto-share. If two people want a shared project (live collab), that needs the Team plan. Otherwise each person sets up their own copy from this page.
