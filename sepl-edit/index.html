📋 Copy-and-paste — the simplest way
4 taps total. Works on any phone/iPad/laptop. Any Claude account (Free or Pro).
Step 1 — pick the page you want to edit
Step 2 — copy everything Claude needs
One tap. Clipboard now has the rules + the page's full HTML source. Ready to paste into any Claude chat.
Step 3 — open Claude and paste
  1. Open the Claude app on your phone/iPad (or claude.ai in a browser).
  2. Start a new chat.
  3. Long-press the message box → Paste.
  4. At the end, type what you want changed (replace the [describe your change here] placeholder). Examples:
    • "Change the Create Intake button text to 'Confirm and Create'."
    • "Make the warning above Create Intake red instead of yellow."
    • "Add a checkbox to mark whether the client brought their PAN card."
  5. Tap Send.
Step 4 — iterate, then handoff
  1. Claude shows the page as a previewable artifact. Tap to see it render full-screen.
  2. Each change you ask for updates the same artifact in place. Watch the preview re-render.
  3. When you're happy, say: "I'm done — handoff."
  4. Claude outputs the final file + a short HANDOFF SUMMARY.
  5. Copy the artifact and the summary, send to Deepak on WhatsApp. He ships it to spicemore.com — live in ~60 seconds.
That's it. No deeplinks, no project setup, no plan upgrade required. Most reliable path because Claude has all the source code right in the message — nothing to fetch, nothing that can fail.

📨 Sending changes back to Deepak
Three ways. Pick whichever is easiest.
  • WhatsApp — copy the artifact text or share the Claude chat URL. Send to Deepak. Fastest.
  • Email — to deepakj62@gmail.com with subject prefix SEPL HANDOFF:.
  • Shared Drive folder — if you've been added to the iCloud / Drive folder, drop the .html file there.

Deepak's Claude Code reads the handoff, sanity-checks against the system rules, commits, and pushes to GitHub. Vercel auto-deploys to spicemore.com. Live in ~60 sec.

What Claude can change here (and what it can't)
Can change: any visual element on the SEPL pages — text, layout, colours within the Spicemore palette, button positions, dropdown options, validation messages, mandatory-field markers, new fields, new sections, new tabs, calculation labels.

Can't change directly (these need Deepak's Claude Code): backend API endpoints, database, scrapers, WhatsApp delivery, OTP, agreement PDF text, LTV bands, margin call thresholds, deployment/infra. If you ask for any of these, Claude will flag "backend change needed — Deepak handles."
Pricing / plan notes
Both paths work on Free, Pro, and Team Claude plans. Free plan has daily message limits (~5–10 messages); Pro is ~5× higher. For day-to-day SEPL editing, Free is enough; for heavy iteration, Pro at $20/month is comfortable.

Claude Projects ARE per-account — they don't auto-share. If two people want a shared project (live collab), that needs the Team plan. Otherwise each person sets up their own copy from this page.
